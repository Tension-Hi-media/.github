# 미니프로젝트

## 💻 `프로젝트 소개 & 개발 동기`
<b>사용자의 채팅 데이터를 기반으로 감정을 분석하여 프로필 상태(예: 화남, 즐거움)를 자동으로 업데이트하거나, 감정에 맞는 이미지를 변경하는 기능을 제공</b>

- 메세지 감정 분석 시스템 구축
    -  메세지에 담겨있는 감정을 분석하고 이에 대한 분석 결과를 emotion과 수치로 나타내도록 했습니다.

- 사용자의 재미와 몰입도를 높이는 기능
    - 나의 기분에 따라 바뀌는 이미지을 통해 대화에 대한 재미와 흥미를 높였습니다.
